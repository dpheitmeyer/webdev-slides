# HTML Basics — Slidev Presentation

An interactive slide deck for beginners learning HTML: elements, tags, attributes, and nesting.

## Requirements

- Node.js >= 20.12

## Getting started

```bash
npm install
npm run dev
```

The presentation opens automatically at `http://localhost:3030`.

## Other commands

```bash
npm run build   # build as a static site
npm run export  # export to PDF
```
